## 📈 How it analyzes stocks

### Technical Indicators

- **RSI**
  - <30 → Oversold → BUY  
  - >70 → Overbought → SELL  

- **MACD**
  - Crossover indicates trend reversal  

- **SMA (20 & 50)**
  - Price above → Bullish  
  - Price below → Bearish  

- **Trend**
  - Based on price vs SMA50  

---

### 📰 Sentiment Analysis

- Fetches latest news articles  
- Detects positive / negative sentiment  
- Helps understand overall market mood

## 📦 Run Locally

```bash
# Clone repository
git clone https://github.com/hafsakhan09090/stockwise-ai.git
cd stockwise-ai

# Create virtual environment
python -m venv venv

# Activate environment
# Windows:
venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Run application
python app.py
```
